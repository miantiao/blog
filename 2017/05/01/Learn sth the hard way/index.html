Learn sth the hard way

发表于 2017-05-01   |   分类于 自思自律

有本书叫《Learn Python the Hard Way》，看过的人应该都知道它之于Python学习意味着什么。我还没看完，但这本书的教育思路再次提醒了我：想学东西，就要找麻烦。

人要成长，就要学习，而学习必然会有路径。有道是：书山有路勤为径，学海无涯苦作舟。蓝鹅，学多了，就会发现勤和苦的表达并不准确。

以前面试应聘者时，经常会听到对方说这样的话：“我虽然没经验，但我肯定会通过加倍的勤奋来弥补经验上的缺失，不怕吃苦，就是我的优势。”态度不错哦，可惜，这话在我这里屁用没有。

勤与苦，只是状态，背后最多也就代表了态度。你愿勤奋，你肯吃苦，所以呢？那你就能在给定的时间内按期望的标准学会指定的东西么？你为了掌握技能而做了些什么才是重点，你准备从哪里开始做才有意义，态度嘛，顶多也就是推动你去做事的动力而已，但动力再足，走错了道就都白走了。

所谓learn sth the hard way，就是用最笨的的办法，从最简单的东西学起。

举一波栗子。练字，就要照着字帖，一笔一划的写。弹琴，就要照着曲谱，一个音一个音的弹。健身，就要跟着教练，一个动作一个动作的练。编程，就要按着示例，一个字节一个字节的敲进电脑。PPP，就要老老实实跟一个项目，一个环节一个环节的做出来。

Learn sth the hard way，不一定要多勤奋，但要准确，不一定要很吃苦，但要坚持，也不一定要搞得很沧桑，但要耐得住寂寞。

在这个讲究速成的年代，找件事情，learn it the hard way，想想就开心。
